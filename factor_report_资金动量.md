# 期货Agent系统 — 资金动量因子精选报告

> **生成日期**: 2026-07-07  
> **数据来源**: `factor_screening_top100.csv` / `.json`  
> **筛选类别**: 资金动量  
> **因子总数**: 24个（原始）→ 8个（精选）  
> **适配目标**: 期货单品种时序信号（Agent系统）

---

## 一、筛选方法论

### 1.1 筛选标准

| 维度 | 阈值 | 说明 |
|------|------|------|
| **质量分 Q** | ≥ 0.85 | 因子稳定性与预测力综合评分 |
| **OOS稳健性 |test_rankicir\|** | ≥ 0.45 | 样本外测试的Rank ICIR绝对值，越高越稳健 |
| **单调性 |monotonicity\|** | ≥ 0.85 | 接近 ±1 表示方向稳定，分组收益单调 |
| **逻辑复杂度** | ≤ 5层 | 排除过度嵌套、难以解释的黑箱因子 |
| **重复性** | 去重 | 排除公式结构雷同、评分指标完全一致的克隆因子 |

### 1.2 从截面到单品种时序的转换原则

原A股因子为**截面选股**设计（即在某一时点，比较全市场股票的因子值，选高/低分组）。本报告将其转换为**单品种时序信号**（即跟踪同一品种自身的因子值变化，判断多空方向）：

- **信号方向**：正ICIR因子 → 因子值高时偏多；负ICIR因子 → 因子值高时偏空（或取反后统一为偏多信号）
- **滚动窗口**：默认20日滚动窗口，适配期货中短期趋势跟踪
- **字段映射**：`开盘价_复权→open_price`、`收盘价_复权→close_price`、`最高价_复权→high_price`、`最低价_复权→low_price`、`成交量→volume`、`成交额→amount`、`前收盘价→pre_close`
- **时序归一化**：所有因子表达式均以 `rolling(N)` 形式呈现，可直接嵌入 pandas 时序计算

## 二、精选因子 Top 8

### 2.1 因子一：开盘收益率动量归一化（AM_ts_mean_return_div_open）

**基础信息**

| 指标 | 值 | 解读 |
|------|------|------|
| **Q** | 0.9487 | ⭐ 质量分Top1，极高稳定性 |
| **test_rankicir** | 0.5679 | 正相关，样本外稳健 |
| **monotonicity** | 0.9 | 正向单调，方向稳定 |
| **ls_sharpe** | 1.1712 | 多空夏普较高 |
| **composite_score** | 0.7738 | 综合评分第一 |
| **所属Pack** | factor_pack_027 |

**因子逻辑（中文解释）**

该因子计算**近期收益率均值**，并以**当前开盘价**进行归一化，同时引入**成交额**作为资金参与度的验证。其核心逻辑是：**若某品种近期积累了正向收益率，且当前开盘价处于相对低位，则未来仍有上涨惯性**。成交额用于过滤低流动性干扰——仅当资金参与活跃时，动量信号才有效。

从交易角度看，这类似于**“动量延续”**策略：当价格已形成上涨趋势，且开盘未出现高开透支涨幅时，趋势更可能延续。归一化处理使得不同价格水平的品种可横向比较。

**A股字段 → 期货字段映射**

| A股字段 | 期货字段 | 说明 |
|---------|----------|------|
| 开盘价_复权 | `open_price` | 日开盘价 |
| 成交额 | `amount` | 日成交额（元） |
| —（隐含） | `close_price` | 计算收益率需收盘价 |
| —（隐含） | `pre_close` | 前收盘价（计算收益率） |

**期货适配 pandas 表达式**

```python
# 计算20日滚动收益率均值
returns = close_price / pre_close - 1
ts_mean_return = returns.rolling(20).mean()

# 用开盘价归一化（开盘价越低，归一化后的动量越强）
raw_factor = ts_mean_return / open_price

# 成交额活跃度过滤（仅考虑成交额高于20日均值的日子）
amount_active = amount > amount.rolling(20).mean()

# 最终因子：未激活时取0，避免低流动性噪音
factor = raw_factor.where(amount_active, 0)

# 信号方向（正ICIR=0.5679）：因子值高 → 偏多；因子值低 → 偏空
signal = np.where(factor > factor.rolling(60).quantile(0.7), 1,
         np.where(factor < factor.rolling(60).quantile(0.3), -1, 0))
```

**交易含义总结**

> **"低价开盘+累积动量"** 信号：当品种在近期已形成正向收益，但当前开盘价并未过度反映（即未高开），且伴随活跃成交时，趋势延续概率较高。反之，若因子值极低（负收益累积+低开），则偏空。这是一个**动量延续型**因子，适合趋势跟踪场景。

### 2.2 因子二：成交额-波动交互（AM_mul_amount_ts_std）

**基础信息**

| 指标 | 值 | 解读 |
|------|------|------|
| **Q** | 0.9134 | 高质量，稳定性强 |
| **test_rankicir** | -0.5205 | 负相关，绝对值高 |
| **monotonicity** | -1.0 | ⭐ 完美负向单调 |
| **ls_sharpe** | -1.1801 | 多空夏普稳健 |
| **composite_score** | 0.7522 | 综合评分高 |
| **所属Pack** | factor_pack_030 |

**因子逻辑（中文解释）**

该因子将**成交额**与**价格波动标准差**相乘，衡量**"资金+波动"的交互强度**。其核心逻辑是：**当大量资金涌入且伴随高波动时，往往意味着市场分歧大、情绪极端，后续更可能反转**。这是一个**负向因子**——交互值越高，未来收益越差。

从交易角度看，成交额放大+波动加剧 = 资金博弈白热化，往往是阶段性顶部或底部的信号。该因子通过乘积放大这种极端状态，比单独看成交量或波动更有效。

**A股字段 → 期货字段映射**

| A股字段 | 期货字段 | 说明 |
|---------|----------|------|
| 最高价_复权 | `high_price` | 日最高价 |
| 最低价_复权 | `low_price` | 日最低价 |
| 收盘价_复权 | `close_price` | 日收盘价 |
| 成交额 | `amount` | 日成交额（元） |
| —（隐含） | `pre_close` | 前收盘价（计算振幅） |

**期货适配 pandas 表达式**

```python
# 计算日内振幅（相对于前收盘价）
amplitude = (high_price - low_price) / pre_close

# 振幅的20日标准差（衡量近期波动强度）
ts_std_amp = amplitude.rolling(20).std()

# 成交额 × 波动标准差 = 资金-波动交互强度
raw_factor = amount * ts_std_amp

# 对数压缩（处理amount量纲差异）+ 20日均值归一化
factor = np.log1p(raw_factor) / np.log1p(raw_factor).rolling(20).mean() - 1

# 信号方向（负ICIR=-0.5205）：因子值高 → 偏空；因子值低 → 偏多
signal = np.where(factor > factor.rolling(60).quantile(0.7), -1,
         np.where(factor < factor.rolling(60).quantile(0.3), 1, 0))
```

**交易含义总结**

> **"资金+波动双高反转"** 信号：当成交额与价格波动同时放大（即资金激烈博弈），市场往往处于极端情绪状态，后续反向运动概率高。这是一个**均值回归型**因子，适合震荡或极端情绪识别场景。因子值高 = 过热需警惕；因子值低 = 冷清或波动收敛，可能酝酿趋势。

### 2.3 因子三：资金流方向-波动偏离（AM_sub_sign_ts_std）

**基础信息**

| 指标 | 值 | 解读 |
|------|------|------|
| **Q** | 0.8951 | 高质量 |
| **test_rankicir** | -0.5067 | 负相关，样本外稳健 |
| **monotonicity** | -0.9 | 负向单调，方向稳定 |
| **ls_sharpe** | -1.6027 | ⭐ 多空夏普极高 |
| **composite_score** | 0.7430 | 综合评分高 |
| **所属Pack** | factor_pack_029 |

**因子逻辑（中文解释）**

该因子衡量**成交额变化方向**与**价格波动**之间的**偏离度**。其核心逻辑是：**当成交额持续放大（方向为正）但价格波动并未同步放大时，说明资金流入是有序的、非恐慌性的，后续更可能上涨**；反之，若成交额萎缩但波动高企，则资金支撑不足，易下跌。

从交易角度看，这是一个**"量价背离"**型因子：健康的上涨需要资金持续流入且波动可控（稳步推高）。如果价格暴涨但资金并未跟进（或资金跟进但价格暴涨过速），则量价背离，趋势不可持续。

**A股字段 → 期货字段映射**

| A股字段 | 期货字段 | 说明 |
|---------|----------|------|
| 成交额 | `amount` | 日成交额（元） |
| 收盘价_复权 | `close_price` | 日收盘价 |

**期货适配 pandas 表达式**

```python
# 成交额相对20日均值的方向（+1=高于均值，-1=低于均值）
sign_amount = np.sign(amount - amount.rolling(20).mean())

# 收盘价的20日变异系数（波动/价格，无量纲）
cv_close = close_price.rolling(20).std() / close_price.rolling(20).mean()

# 偏离度：资金方向 - 价格波动（方向为正但波动低 → 因子高）
raw_factor = sign_amount - cv_close

# 平滑处理（5日EMA）
factor = raw_factor.ewm(span=5).mean()

# 信号方向（负ICIR=-0.5067）：因子值高 → 偏空；因子值低 → 偏多
# 注意：这里"高"指 sign_amount - cv_close 大，即资金流入但波动极低，
# 原A股为负ICIR，意味着期货中应取反向
# 更直观的理解：因子值高 = 资金方向强但波动低 = 健康上涨 → 应偏多
# 因此建议取反：使用 -factor 作为信号
signal = np.where((-factor) > (-factor).rolling(60).quantile(0.7), 1,
         np.where((-factor) < (-factor).rolling(60).quantile(0.3), -1, 0))
```

**交易含义总结**

> **"量价健康度"** 信号：当资金持续流入（成交额高于均值）而价格波动率较低时，说明上涨是有序的、资金支撑扎实的，趋势可延续。反之，若价格波动率高但资金流入不足，则是"虚涨"，需警惕回落。这是一个**趋势质量型**因子，ls_sharpe高达1.60，多空区分能力极强。

### 2.4 因子四：成交额变化稳定性（AM_ts_inverse_cv_delta_amount）

**基础信息**

| 指标 | 值 | 解读 |
|------|------|------|
| **Q** | 0.9010 | 高质量 |
| **test_rankicir** | -0.4893 | 负相关，绝对值高 |
| **monotonicity** | -0.9 | 负向单调，方向稳定 |
| **ls_sharpe** | -1.3107 | 多空夏普稳健 |
| **composite_score** | 0.7310 | 综合评分高 |
| **所属Pack** | factor_pack_028 |

**因子逻辑（中文解释）**

该因子计算**成交额变化的逆变异系数**（Inverse Coefficient of Variation），即 **均值/标准差**。其核心逻辑是：**当成交额的变化呈现稳定、可预测的模式时（逆变异系数高），往往意味着资金行为趋于一致，后续更可能反转**；反之，若成交额变化杂乱无章（逆变异系数低），则市场处于无序状态。

逆变异系数（ICV）是变异系数（CV=std/mean）的倒数。ICV高 = 变化稳定（均值大且波动小）；ICV低 = 变化不稳定（均值小或波动大）。该因子为负向，即**成交额变化越稳定，未来收益越差**——稳定资金流入后容易力竭。

**A股字段 → 期货字段映射**

| A股字段 | 期货字段 | 说明 |
|---------|----------|------|
| 成交额 | `amount` | 日成交额（元） |

**期货适配 pandas 表达式**

```python
# 成交额日变化（一阶差分）
delta_amount = amount.diff()

# 20日滚动均值和标准差
mean_delta = delta_amount.rolling(20).mean()
std_delta = delta_amount.rolling(20).std()

# 逆变异系数（均值/标准差），并处理除零
raw_factor = mean_delta / std_delta.replace(0, np.nan)

# 填充缺失值并平滑
factor = raw_factor.fillna(0).rolling(5).mean()

# 信号方向（负ICIR=-0.4893）：因子值高 → 偏空；因子值低 → 偏多
# 逻辑：成交额变化过于稳定 → 资金动能衰减 → 反向
signal = np.where(factor > factor.rolling(60).quantile(0.7), -1,
         np.where(factor < factor.rolling(60).quantile(0.3), 1, 0))
```

**交易含义总结**

> **"资金动能衰减"** 信号：当成交额的变化呈现规律性强、波动小的特征时（如每日稳定放量），说明资金流入已进入"疲劳期"，后续难以维持，趋势反转概率高。反之，成交额变化杂乱无章（资金进出无序），市场可能仍在酝酿方向。这是一个**动能衰竭型**因子，适合识别趋势末端。

### 2.5 因子五：成交额动量叠加（AM_ts_pct_change_add_amount）

**基础信息**

| 指标 | 值 | 解读 |

| **Q** | 0.8976 | 高质量 |
| **test_rankicir** | -0.5060 | 负相关，绝对值高 |

| **composite_score** | 0.7219 | 综合评分高 |

**因子逻辑（中文解释）**

该因子将**成交额的变化率**与**成交额相对均值的偏离度**叠加，衡量**成交额动量的双重确认**。其核心逻辑是：**当成交额同时满足"短期变化率大"和"高于历史均值"两个条件时，资金动量达到极端，后续易反转**。

从交易角度看，这是**成交额动量的极端值识别**：pct_change衡量短期爆发力，add_amount衡量相对历史的位置。两者同时高 = 成交额处于"超买"状态，资金热情过盛，后续冷却概率大。

**A股字段 → 期货字段映射**

| A股字段 | 期货字段 | 说明 |
|---------|----------|------|
| 成交额 | `amount` | 日成交额（元） |

**期货适配 pandas 表达式**

```python
# 成交额20日变化率
pct_change = amount.pct_change(20)

# 成交额相对20日均值的偏离度（归一化）
deviation = amount / amount.rolling(20).mean() - 1

# 叠加：变化率 + 均值偏离度（双重动量确认）
raw_factor = pct_change + deviation

# 20日滚动平滑，降低噪音
factor = raw_factor.rolling(20).mean()

# 信号方向（负ICIR=-0.5060）：因子值高 → 偏空；因子值低 → 偏多
# 逻辑：成交额动量双高 → 过热 → 反向
signal = np.where(factor > factor.rolling(60).quantile(0.7), -1,
         np.where(factor < factor.rolling(60).quantile(0.3), 1, 0))
```

**交易含义总结**

> **"成交额动量极端"** 信号：当成交额短期暴涨（pct_change高）且处于历史高位（偏离均值大）时，资金参与度过热，后续冷却回落的概率高。反之，成交额持续低迷（双低）时，可能处于底部蓄势阶段。这是一个**成交量超买超卖型**因子，类似成交量的RSI概念。

### 2.6 因子六：对数成交额增长率（AM_log_ts_pct_change_amount）

**基础信息**

| 指标 | 值 | 解读 |
|------|------|------|
| **Q** | 0.8884 | 高质量 |
| **test_rankicir** | -0.4947 | 负相关，绝对值高 |
| **monotonicity** | -0.9 | 负向单调，方向稳定 |
| **ls_sharpe** | -1.1548 | 多空夏普稳健 |
| **composite_score** | 0.7211 | 综合评分高 |
| **所属Pack** | factor_pack_026 |

**因子逻辑（中文解释）**

该因子计算**成交额的对数增长率**，即 `log(当前成交额 / 20日前成交额)`。对数变换的核心优势是**压缩极端值**：当成交额暴增10倍时，线性增长率会达到900%，而对数增长率仅约2.3，使得极端值不会过度扭曲信号。

其交易逻辑是：**成交额的对数增长越快，说明资金涌入速度越极端，后续反转概率越高**。对数变换让该因子对"温和放量"和"天量成交"的区分更加敏感——温和放量可能伴随趋势延续，而天量对数增长则明确指向过热。

**A股字段 → 期货字段映射**

| A股字段 | 期货字段 | 说明 |
|---------|----------|------|
| 成交额 | `amount` | 日成交额（元） |

**期货适配 pandas 表达式**

```python
# 对数成交额增长率（20日）
# 等价于 np.log(amount / amount.shift(20))，天然规避负值问题
raw_factor = np.log(amount / amount.shift(20))

# 处理负值和零值（期货可能有无成交时段）
raw_factor = raw_factor.replace([-np.inf, np.inf], np.nan).fillna(0)

# 20日滚动平滑
factor = raw_factor.rolling(20).mean()

# 信号方向（负ICIR=-0.4947）：因子值高 → 偏空；因子值低 → 偏多
# 逻辑：对数增长过快 → 资金过热 → 反向
signal = np.where(factor > factor.rolling(60).quantile(0.7), -1,
         np.where(factor < factor.rolling(60).quantile(0.3), 1, 0))
```

**交易含义总结**

> **"资金增速压缩视角"** 信号：通过对数变换压缩成交额极端值，更准确地识别"温和积累"与"极端爆发"的区别。对数增长率的优点是，10倍放量和20倍放量在对数空间中的差距被拉平，不会因为个别天量数据而过度扭曲。适合作为**成交量过滤条件**，与价格动量因子搭配使用。

### 2.7 因子七：成交额极差+短期收益（AM_ts_maxmin_add_ret_5）

**基础信息**

| 指标 | 值 | 解读 |
|------|------|------|
| **Q** | 0.8853 | 高质量 |
| **test_rankicir** | -0.4621 | 负相关，绝对值高 |
| **monotonicity** | -1.0 | ⭐ 完美负向单调 |
| **ls_sharpe** | -1.0028 | 多空夏普稳健 |
| **composite_score** | 0.7147 | 综合评分高 |
| **所属Pack** | factor_pack_024 |

**因子逻辑（中文解释）**

该因子将**成交额的20日极差**（最大值-最小值）与**5日收益率**叠加，衡量**"资金波动范围+短期收益"的复合极端度**。其核心逻辑是：**当成交额在近期经历了从极低到极高的宽幅波动，且同时伴随短期正收益时，说明资金先试探后涌入，阶段性高点已至**。

从交易角度看，成交额极差大 = 资金态度从冷淡到热情的剧烈转变，这种转变往往不可持续；叠加5日正收益 = 价格已部分反映资金涌入，后续追高性价比低。这是一个**"资金态度转变+价格已反应"**的复合反转信号。

**A股字段 → 期货字段映射**

| A股字段 | 期货字段 | 说明 |
|---------|----------|------|
| 成交额 | `amount` | 日成交额（元） |
| 收盘价_复权 | `close_price` | 日收盘价 |
| —（隐含） | `pre_close` | 前收盘价（计算5日收益） |

**期货适配 pandas 表达式**

```python
# 成交额20日极差（归一化到均值）
maxmin_amount = (amount.rolling(20).max() - amount.rolling(20).min()) / amount.rolling(20).mean()

# 5日收益率
ret_5 = close_price / close_price.shift(5) - 1

# 叠加：资金极差 + 短期收益（均无量纲，可加）
raw_factor = maxmin_amount + ret_5

# 10日EMA平滑
factor = raw_factor.ewm(span=10).mean()

# 信号方向（负ICIR=-0.4621）：因子值高 → 偏空；因子值低 → 偏多
# 逻辑：资金从冷到热+短期已涨 → 阶段性顶点
signal = np.where(factor > factor.rolling(60).quantile(0.7), -1,
         np.where(factor < factor.rolling(60).quantile(0.3), 1, 0))
```

**交易含义总结**

> **"资金态度转变顶点"** 信号：当成交额在近期经历了从地量到天量的宽幅波动（极差大），且价格已录得短期正收益时，说明资金从观望到狂热的过程已经完成，后续接盘力量不足。这是一个**阶段性顶部识别**因子，monotonicity=-1.0表示完美单调，信号可靠性极高。

### 2.8 因子八：DEMA-成交额交互（AM_ts_dema_mul_amount）

**基础信息**

| 指标 | 值 | 解读 |
|------|------|------|
| **Q** | 0.8963 | 高质量 |
| **test_rankicir** | -0.4882 | 负相关，绝对值高 |
| **monotonicity** | -1.0 | ⭐ 完美负向单调 |
| **ls_sharpe** | -1.3257 | 多空夏普稳健 |
| **composite_score** | 0.7388 | 综合评分高 |
| **所属Pack** | factor_pack_026 |

**因子逻辑（中文解释）**

该因子将**DEMA（双指数移动平均）**与**成交额**相乘，衡量**"趋势强度×资金参与"的交互效应**。DEMA比EMA对近期价格更敏感，能更快捕捉趋势转折。核心逻辑是：**当DEMA信号强烈（趋势加速）且成交额放大时，趋势已进入加速阶段，后续反转概率高**。

从交易角度看，DEMA×成交额相当于"趋势加速度×质量"——如果趋势加速但资金未跟进，是虚涨；如果趋势加速且资金涌入，是趋势末端的"最后疯狂"。该因子为负向，捕捉的是**"趋势加速+资金涌入"的极端状态**。

**A股字段 → 期货字段映射**

| A股字段 | 期货字段 | 说明 |
|---------|----------|------|
| 最高价_复权 | `high_price` | 日最高价 |
| 最低价_复权 | `low_price` | 日最低价 |
| 收盘价_复权 | `close_price` | 日收盘价 |
| 前收盘价 | `pre_close` | 前收盘价 |
| 成交额 | `amount` | 日成交额（元） |

**期货适配 pandas 表达式**

```python
# 计算收盘价DEMA（20日，双指数移动平均）
# DEMA = 2*EMA - EMA(EMA)
ema1 = close_price.ewm(span=20).mean()
ema2 = ema1.ewm(span=20).mean()
dema = 2 * ema1 - ema2

# 计算日内振幅（相对于前收盘价）
amplitude = (high_price - low_price) / pre_close

# DEMA变化率（衡量趋势加速度）
dema_change = dema / dema.shift(1) - 1

# 成交额 × DEMA变化率 × 振幅 = 资金×趋势加速度×波动
raw_factor = amount * dema_change * amplitude

# 对数压缩并平滑
factor = np.log1p(raw_factor).rolling(10).mean()

# 信号方向（负ICIR=-0.4882）：因子值高 → 偏空；因子值低 → 偏多
# 逻辑：趋势加速+资金涌入+波动大 = 最后疯狂 → 反向
signal = np.where(factor > factor.rolling(60).quantile(0.7), -1,
         np.where(factor < factor.rolling(60).quantile(0.3), 1, 0))
```

**交易含义总结**

> **"趋势加速末端"** 信号：DEMA对趋势转折比EMA更敏感，当DEMA显示趋势加速、且成交额同步放大、振幅扩大时，三者共同构成"趋势末端三重确认"。这是经典的**"加速即结束"**交易逻辑——最猛烈的上涨往往发生在最后一波。该因子monotonicity=-1.0，完美负向单调，适合作为**趋势末端反转**的预警指标。

    # 计算各因子（按上述表达式）

    # 加权合成

    composite = (

## 五、风险提示与局限性

1. **数据频率**: 以上因子基于日K线设计，若用于分钟级交易，需相应缩短滚动窗口（如20日→20小时）
2. **品种差异**: 不同期货品种（股指/商品/国债）的成交额特征差异大，建议分品种群组标准化
3. **换月影响**: 期货合约换月时成交量和成交额可能剧烈变化，建议用**连续合约**或做**换月调整**
4. **极端行情**: 在涨停/跌停或流动性枯竭时，成交额因子可能失真，需配合价格因子交叉验证
5. **因子衰减**: 自动挖掘因子存在**过拟合风险**，建议在小样本品种上先回测验证，再逐步推广
