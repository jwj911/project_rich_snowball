### 因子 1：AM_ts_mean_return_div_open_d4085de9
**（资金动量 → 价格动量逻辑）**

| 指标 | 数值 | 说明 |
|------|------|------|
| Q | **0.9487** | 极高稳定性 |
| test_rankicir | **0.5679** | OOS 强预测力 |
| monotonicity | **0.90** | 正向单调，方向稳定 |
| ls_sharpe | **1.1712** | 多空夏普优秀 |
| A股原始字段 | 开盘价_复权、成交额 | 期货映射见下表 |

#### 因子逻辑
过去一段时间的平均收益率除以当日开盘价，衡量**单位开盘价承载的平均收益效率**。本质上是一个**价格趋势强度**信号：当价格趋势向上且开盘价相对较低时，该因子值较大，预示后续价格可能延续上涨。成交额在原始列中，但核心驱动为价格收益部分。

#### 适配期货的 Pandas 表达式

```python
# 20日平均收益率 / 当日开盘价
daily_ret = close_price.pct_change()
mean_ret_20 = daily_ret.rolling(window=20).mean()
factor = mean_ret_20 / open_price
```

#### 信号方向
- **factor > 0 且上升**：做多信号（趋势向上，收益效率高）
- **factor < 0 且下降**：做空信号（趋势向下，收益效率为负）

### 因子 2：AM_ema_ts_dema_delta_8dc6d97c
**（波动振幅 → 趋势加速动量）**

| 指标 | 数值 | 说明 |
|------|------|------|
| Q | **0.8826** | 高稳定性 |
| test_rankicir | **0.4554** | OOS 稳健正向 |
| monotonicity | **1.00** | 完美正向单调 |
| ls_sharpe | **1.5643** | 多空夏普极佳 |
| A股原始字段 | 最高价_复权、前收盘价、最低价_复权 | |

#### 因子逻辑
先计算**DEMA（双重指数移动平均）的日度变化量**（即 DEMA 的加速度），再对其进行 **EMA 平滑**，得到**趋势加速度信号**。DEMA 对价格变化的反应比 EMA 更灵敏，其日度变化量反映趋势加速/减速；再经 EMA 平滑后，过滤噪声，保留中周期趋势方向。该因子为**纯价格技术指标**，不含成交量/成交额，适配期货最直接。

#### 适配期货的 Pandas 表达式

```python
# 典型价格（Typical Price）
tp = (high_price + low_price + pre_close) / 3

# EMA 平滑
ema_tp = tp.ewm(span=12, adjust=False).mean()

# DEMA = 2*EMA - EMA(EMA)
dema_tp = 2 * ema_tp - ema_tp.ewm(span=12, adjust=False).mean()

# DEMA 的日度变化（加速度）
dema_delta = dema_tp.diff()

# 对加速度再做 EMA 平滑，得到最终因子
factor = dema_delta.ewm(span=12, adjust=False).mean()
```

#### 信号方向
- **factor > 0 且扩大**：趋势加速向上，做多信号
- **factor < 0 且扩大**：趋势加速向下，做空信号
- **factor 趋近于 0**：趋势减速或震荡，观望

### 因子 3：AM_ts_regression_beta_intraday_rang_b4142f62
**（波动振幅 → 振幅-收益回归动量）**

| 指标 | 数值 | 说明 |
|------|------|------|
| Q | **0.8790** | 高稳定性 |
| test_rankicir | **-0.4525** | OOS 稳健负向 |
| monotonicity | **-0.90** | 负向单调，方向稳定 |
| ls_sharpe | **-1.4432** | 多空夏普优秀 |
| A股原始字段 | 最高价_复权、收盘价_复权、最低价_复权 | |

#### 因子逻辑
对过去 N 个交易日的**日内振幅**（`(high - low) / close`）与**日收益率**进行滚动线性回归，提取回归斜率 **beta**。该因子衡量「价格对波动的敏感度」：beta 越大，说明同样的振幅往往对应更大的收益，意味着价格动量对波动有更强的传导效应。在期货时序中，beta 为负表示**高波动对应低收益**（波动率陷阱），可用于识别假突破。

#### 适配期货的 Pandas 表达式

```python
# 日内振幅（Intraday Range）
intraday_range = (high_price - low_price) / close_price

# 日收益率
daily_ret = close_price.pct_change()

# 滚动回归 beta（window=20）
window = 20
cov = intraday_range.rolling(window).cov(daily_ret)
var = intraday_range.rolling(window).var()
factor = cov / var
```

#### 信号方向
- **factor < 0 且绝对值扩大**：高波动伴随低收益，做空或平仓信号（波动率陷阱）
- **factor > 0**：高波动伴随高收益，趋势确认，做多信号

### 因子 4：AM_ts_min_neg_ret_20_8ece5d20
**（其他 → 下行抗跌动量）**

| 指标 | 数值 | 说明 |
|------|------|------|
| Q | **0.8953** | 高稳定性 |
| test_rankicir | **0.4761** | OOS 稳健正向 |
| monotonicity | **0.90** | 正向单调 |
| ls_sharpe | **0.8775** | 多空夏普良好 |
| A股原始字段 | 收盘价_复权 | 纯价格字段 |

#### 因子逻辑
过去 **20 个交易日**中，所有**负收益日**的**最小值**（即最差单日跌幅）。该因子为**正**时表示：过去 20 日最差的一次下跌并不深，说明价格**抗跌性强、支撑有效**。在期货时序中，这是一个**下行风险保护**信号：当该因子从低位回升（最差跌幅变浅），暗示空头力量衰竭，可能迎来反弹或反转。

#### 适配期货的 Pandas 表达式

```python
# 日收益率
daily_ret = close_price.pct_change()

# 只保留负收益（正值日截断为0）
neg_ret = daily_ret.clip(upper=0)

# 过去20日最小负收益（即最差跌幅）
factor = neg_ret.rolling(window=20).min()
```

#### 信号方向
- **factor 从极负值回升**（如 -0.05 → -0.02）：下行风险收敛，空头衰竭，关注做多机会
- **factor 持续为 0**：过去 20 日无下跌，价格极强，但需警惕回调
- **factor 创出新低**（更负）：下行风险加剧，做空或止损信号

### 因子 5：AM_ts_max_ret_20_296b1c32
**（其他 → 上行突破动量）**

| 指标 | 数值 | 说明 |
|------|------|------|
| Q | **0.8930** | 高稳定性 |
| test_rankicir | **-0.4726** | OOS 稳健负向 |
| monotonicity | **-0.90** | 负向单调 |
| ls_sharpe | **-0.9258** | 多空夏普良好 |
| A股原始字段 | 收盘价_复权 | 纯价格字段 |

#### 因子逻辑
过去 **20 个交易日**中的**最大单日收益**。该因子捕捉价格的**短期爆发力**。在 A 股截面中，该因子为负（mono=-0.9），意味着**过去涨得越猛，未来越可能回调**（动量反转）。在期货时序中，可直接用作**反转信号**：当 20 日最大收益创出极值时，警惕价格过度延伸，考虑做空或止盈。

#### 适配期货的 Pandas 表达式

```python
# 日收益率
daily_ret = close_price.pct_change()

# 过去20日最大单日收益
factor = daily_ret.rolling(window=20).max()
```

#### 信号方向
- **factor 创出极高正值**：价格过度延伸，动量衰竭，做空/止盈信号
- **factor 处于低位**：价格缺乏爆发力，观望或等待突破
- **factor 与当前价格背离**：最大收益在缩小但价格仍在涨，顶背离预警

### 因子 6：AM_ts_max_clip_ret_20_02db363e
**（价格动量 — Top100 中唯一标签）**

| 指标 | 数值 | 说明 |
|------|------|------|
| Q | **0.8951** | 高稳定性 |
| test_rankicir | **-0.4463** | OOS 稳健（略低于 0.45，但为唯一标签因子） |
| monotonicity | **-1.00** | 完美负向单调 |
| ls_sharpe | **-0.7903** | 多空夏普良好 |
| A股原始字段 | 收盘价_复权 | 纯价格字段 |

#### 因子逻辑
过去 **20 日收益经截断（clip）后的最大值**。通过将日收益截断到合理区间（如 ±5%），**抑制极端异常值**对信号的干扰，使因子更稳健。该因子在 A 股截面中为负（mono=-1.0），意味着**截断后的最大收益越高，未来越可能反转**。在期货时序中，作为**稳健版动量反转**信号：当截断后最大收益持续处于高位，价格可能已过度乐观，应警惕回调。

#### 适配期货的 Pandas 表达式

```python
# 日收益率
daily_ret = close_price.pct_change()

# 截断到 ±5%（抑制极端收益/暴跌）
clip_ret = daily_ret.clip(lower=-0.05, upper=0.05)

# 过去20日截断收益的最大值
factor = clip_ret.rolling(window=20).max()
```

#### 信号方向
- **factor 接近上限（0.05）**：价格连续强势但已被截断，真实动量可能更强，需结合未截断版本确认
- **factor 从高位回落**：截断后的爆发力减弱，趋势可能衰竭
- **factor 处于低位（负值或接近0）**：价格缺乏动能，观望

### 3.1 核心属性对比

| 因子 | 类别来源 | 核心逻辑 | 纯价格 | Q | \|test_rankicir\| | Mono | 方向 |

| AM_ts_regression_beta_intraday_rang | 波动振幅 | 振幅-收益敏感度 | ✅ | 0.8790 | 0.4525 | -0.90 | 波动陷阱 |

### 3.2 逻辑互补性

- **趋势组**（正向）：`mean_return_div_open` + `ema_ts_dema_delta` → 捕捉**中周期趋势方向**
- **反转组**（负向）：`max_ret_20` + `max_clip_ret_20` → 捕捉**短期动量衰竭**
- **风险组**（双向）：`min_neg_ret_20` + `regression_beta_intraday_rang` → 捕捉**下行风险收敛**与**波动陷阱**

### 3.3 期货 Agent 集成建议

在期货 Agent 系统中，建议按以下方式使用：

1. **作为时序信号输入**：每个因子计算为一个时间序列，每日更新。
2. **信号合成**：
   - 正向因子（mono > 0）：标准化后取正权重
   - 负向因子（mono < 0）：标准化后取负权重（或直接反转）
3. **合成公式示例**：

```python
# 单品种时序信号合成（等权法）
signal = (
    zscore(factor_mean_return_div_open) * 0.20 +
    zscore(factor_ema_dema_delta) * 0.20 +
    zscore(factor_regression_beta) * (-0.15) +   # 负向因子反转
    zscore(factor_min_neg_ret) * 0.15 +
    zscore(factor_max_ret) * (-0.15) +          # 负向因子反转
    zscore(factor_max_clip_ret) * (-0.15)       # 负向因子反转
)
```

4. **触发阈值**：
   - `signal > 1.5` → 强做多
   - `signal < -1.5` → 强做空
   - `-0.5 < signal < 0.5` → 观望/平仓

## 四、风险提示与局限性

1. **test_rankicir 略低于 0.45**：`AM_ts_max_clip_ret_20` 的 \|test_rankicir\| = 0.4463，略低于严格阈值，但因其是 Top100 中唯一标签为「价格动量」的因子，且 mono = -1.0 完美，故保留作为代表性因子。

2. **成交额依赖**：`AM_ts_mean_return_div_open` 原始列包含成交额，但核心逻辑为价格收益。若需**纯价格因子**，可排除该因子，仅使用前 5 个纯价格因子。

3. **参数敏感性**：`window=20` 和 `span=12` 为常见默认参数，实际应用中需根据期货品种（如商品期货 vs 股指期货）的波动特性进行调参。

4. **截面→时序转换**：原始因子在 A 股中为截面信号（横截面排序），适配到期货时需转换为**单品种时序信号**（即该品种自身的因子值作为多空判断），已在本报告中完成转换。
